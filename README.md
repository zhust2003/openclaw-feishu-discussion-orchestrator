# OpenClaw Feishu Discussion Orchestrator Skill

一个面向飞书群聊的 OpenClaw 协调者技能。

这个 skill 的目标是让一个协调者机器人在飞书群里：

- 自动发现其他机器人协作者
- 默认并行发起讨论
- 对每个协作者再发起一轮简短质疑
- 最终汇总多方观点，给出综合结论

## 特点

- 只面向飞书群聊
- 默认中文输出
- 默认多协作者并行讨论
- 找不到协作者时自动降级为直接回答
- 不暴露内部 prompt、skill、协议等实现细节

## 文件

- `SKILL.md`：技能定义

## 适用场景

- 主持多机器人群聊讨论
- 内容选题讨论
- 决策建议汇总
- 让一个协调者汇总多个专家机器人意见

## 当前状态

这是一个偏实战导向的技能原型，适合继续迭代，不承诺零配置即用。
